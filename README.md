# TelegramSnitch

Setup the `BOT_TOKEN` in the `.env` file.

## Installation

Execute:

    $ bundle install

## Usage

Run the bot: 

    $ bundle exec bin/bot

Run the listner:

    $ bundle exec thin start

Run the publisher:

    $ bundle exec bin/publisher
